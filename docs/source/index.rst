Dokumentasi Edoo Dashboard
===================================

.. meta::
   :description lang=en: Automate building, versioning, and hosting of your technical documentation continuously on Read the Docs.


**Edoo Dashboard** adalah aplikasi dashboard dari platform Edoo, dimana pada edoo dashboard akan disediakan menu-menu management dari aplikasi Edoo itu sendiri.

**Edoo Dashboard** sendiri memiliki beberapa jenis role akses; Role internal dan role akademik.

Untuk dapat memulai mendevelop edoo dashboard, programmer diharuskan memiliki spesifikasi sebagai berikut : ::

    - Memahami Javascript dan Typescript  
    - Memahami HTML dan CSS (Sass)
    - Memahami React Js
    - Memahami Next Js

Sangat dianjurkan untuk mempelajari javascript dari link berikut : `JavaScript <https://developer.mozilla.org/en-US/docs/Web/JavaScript?retiredLocale=id>`_

Sedangakan untuk mempelajari HTML bisa melalui link berikut : `HTML <https://developer.mozilla.org/en-US/docs/Web/HTML>`_

Kemudian untuk CSS dapat dipelajari dari link berikut: `CSS <https://developer.mozilla.org/en-US/docs/Web/CSS>`_

Setelah 3 komponent tersebut telah dikuasai, maka pelajarilah `ReactJs <https://reactjs.org/docs/getting-started.html>`_
dan `NextJs <https://nextjs.org/>`_
